# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-XX

### Added

#### CLI Tool
- Initial CLI with `init`, `add`, and `list` commands
- Interactive integration selection
- Automatic dependency installation
- Environment variable management
- Support for npm, pnpm, and yarn

#### Integration Templates
- **Stripe**: Payment processing with webhooks and checkout
- **Clerk**: Authentication with middleware and user management
- **Resend**: Email sending with React Email templates
- **Liveblocks**: Real-time collaboration with presence and cursors
- **Supabase**: Database with auth and real-time subscriptions

#### Marketing Website
- Homepage with feature showcase
- Pricing page with monthly and lifetime plans
- Integrations catalog
- Documentation pages
- User dashboard

#### Backend
- Stripe checkout integration
- Webhook handlers for payments
- Supabase database schema
- License key generation and management

### Features

- Production-ready code with error handling
- TypeScript support throughout
- Comprehensive documentation for each integration
- Beautiful UI with Tailwind CSS and shadcn/ui
- Automatic license key generation on purchase
- Support for both one-time and subscription payments
